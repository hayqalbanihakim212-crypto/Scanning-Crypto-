## ✨ Fitur Utama

- 🔎 **Scanning Kripto** — Analisis aset kripto secara mendalam
- ☪️ **Filter Syariah** — Penyaringan berdasarkan prinsip halal & haram
- 📊 **Dashboard Interaktif** — Visualisasi data yang informatif
- ⚡ **Real-time Data** — Informasi harga dan status terkini

---

## 🛠️ Teknologi

| Layer | Teknologi |
|-------|-----------|
| Frontend | React + Vite + Tailwind CSS |
| Backend | REST API (port `8080`) |
| Build Tool | Vite |

---

## 🚀 Cara Menjalankan

### Prerequisites
- Node.js `v18+`
- npm atau yarn

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
# sesuaikan dengan runtime backend kamu
```

Akses di: `http://localhost:5173`
